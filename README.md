# Wind Turbine Blade Acoustic Dataset for Damage Detection

## Overview

This repository contains an acoustic dataset collected for the diagnosis of wind turbine blade faults using deep learning and signal processing techniques. The dataset includes raw acoustic recordings, processed signals, extracted features, and labels.

## Dataset Description

The dataset consists of acoustic signals recorded from a laboratory-scale horizontal-axis wind turbine under different blade conditions.

Classes:

- Healthy
- Crack
- Erosion
- Bend
- Corrugation

## Experimental Setup

- Wind turbine type: Horizontal-axis
- Number of blades: 3
- Microphone: PCB 130F20
- Sampling frequency: 50 kHz
- Recording duration: 2 s
- Environment: Outfield
